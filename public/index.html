The Architecture Blueprint

Discover how the local data pipelines extract and persist items down to the local file layout securely.

Authentication Layer

Manages login sessions safely using Expo SecureStore hooks coupled to cookie intercepts inside native WebViews.

Sync Engine

Coordinates pagination indexing workflows using state machines built entirely inside a lightweight Zustand framework.

WatermelonDB Layer

High-speed relational data abstraction executing fast bulk writes straight to local storage modules over JSI bridges.

Instagram API Client

Assembles custom web-compliant browser wrapper structures injecting secure user-agents, tokens, and verification parameters.

Media Extraction Layer

Normalizes messy incoming JSON payloads into tightly typed, pristine file asset records matching app schema keys.

Expo Router Layout

State-aware link configurations controlling view lifecycle setups, routing states, and modal structures safely.

Synchronization Lifecycle

1

Session Verification Routine

App bootstraps, pulls hidden tokens from system storage, and executes a test inquiry to confirm valid session contexts.

2

Inbox Crawl Phase

Fetches standard profile lists recursively while processing pagination metadata and storing state cursors locally.

3

Node Target Parse Flow

Navigates individual chat rows, processes content arrays, and saves clean media structures while dropping duplicate object records.

Production Safe

Zero Cloud Footprint. Total Control.

Thread Vault operates entirely inside an isolated execution zone. Your access credentials never travel to external servers outside your chosen client target.

Hardware Encryption Bind

All extraction keys remain encrypted on-device via SecureStore blocks.

Logging out invokes a destructive drop cycle resetting the system database storage clean immediately.
